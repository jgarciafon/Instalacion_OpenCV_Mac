# Instalacion_OpenCV_Mac
# 1. Instalar Xcode
```
sudo xcodebuild -license
sudo xcode-select --install
```
# 2. Install Homebrew
```
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
brew update
nano ~/.bash_profile
```
